# Documentazione

Guide di riferimento per gli strumenti e i workflow usati nel progetto. Ogni
file copre uno scope specifico e può essere letto in modo indipendente.

## Indice

- [`gh-cli.md`](./gh-cli.md) — GitHub CLI: PR, issue, workflow, release.
- [`git-workflow.md`](./git-workflow.md) — Conventional Commits, husky,
  commitlint, branching e gestione dei commit.
- [`dependabot.md`](./dependabot.md) — configurazione Dependabot, comandi
  interattivi, strategia di merge.
- [`release-please.md`](./release-please.md) — versioning automatico e changelog
  generati dai Conventional Commits.
- [`astro-stack.md`](./astro-stack.md) — panoramica dello stack: Astro, Tailwind
  v4, TypeScript strict, Vitest, Playwright, Knip.
- [`ci.md`](./ci.md) — pipeline GitHub Actions e quality gate locali.
